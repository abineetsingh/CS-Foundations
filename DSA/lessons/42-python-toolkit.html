The Python Interview Toolkit

Today's skill: the standard-library kit that makes interview Python fast to write and pleasant to read — plus the three gotchas that bite live.

Why this, now

Under time pressure, fluency is speed: knowing Counter exists saves five minutes; knowing the mutable-default gotcha saves a live debugging spiral. This lesson consolidates tools you've already met and fills the gaps. Its companion reference sheet is built for the night before an interview.

The big four imports (old friends, one place)

from collections import Counter, defaultdict, deque
import heapq
import bisect                     # Lesson 14's leftmost(), productionized

Counter("hello")                  # {'l': 2, 'h': 1, …} + .most_common(k)
defaultdict(list)                 # adjacency lists (28); defaultdict(int) counters
deque()                           # O(1) both ends — queues/BFS (11, 20)
heapq.heappush / heappop          # priority queues (23)
bisect.bisect_left(sorted_a, x)   # binary search in one call (14)

Idioms that compress interview code

for i, x in enumerate(nums): ...            # index + value, never range(len())
for a, b in zip(xs, ys): ...                # parallel iteration
if all(c.isdigit() for c in s): ...         # all()/any() over a generator
best = max(intervals, key=lambda iv: iv[1]) # max/min with a key
squares = {x: x*x for x in nums}            # dict/set comprehensions
a, b = b, a                                 # swap — no temp (reverse_list, 13!)
matrix = [[0] * cols for _ in range(rows)]  # 2-D init — see gotcha #2
list(zip(*matrix))                          # transpose (rotate-image problems)
divmod(x, 10)                               # (x // 10, x % 10) — digit problems
float("inf")                                # safe sentinel for min-hunts (37)

The three gotchas that bite in interviews

1. Mutable default arguments

def bfs(start, visited=set()):      # ONE set, shared across ALL calls
    ...
# second test case inherits the first's visited set → wrong answers.
def bfs(start, visited=None):       # the standard fix
    visited = visited if visited is not None else set()

Defaults are evaluated once at definition. (Lesson 34's memo={} exploited this deliberately — as a shared cache it's a feature; as shared state between test cases it's a bug. Know which you're doing.)

2. [[0] * cols] * rows — aliased rows

grid = [[0] * 3] * 2                # TWO references to ONE row
grid[0][0] = 9                      # → [[9, 0, 0], [9, 0, 0]]  !!
grid = [[0] * 3 for _ in range(2)]  # fresh row each time ✓

The outer * copies references (Lesson 12's pointers, biting back). Every 2-D DP table (36) starts here — get it right reflexively.

3. Shallow copy of nested structures

path.copy() / path[:]     # copies the LIST, not lists inside it
# fine for backtracking's flat paths (26);
# for nested boards: import copy; copy.deepcopy(board) — O(cells), say the cost.

Sorting recap in three lines (it's that central)

arr.sort()                                   # in place, returns None (16!)
sorted(pairs, key=lambda p: (p[0], -p[1]))   # tuple keys: X asc, Y desc
merged.sort(key=lambda iv: iv[0])            # intervals: by start (39)
Mission link · ML/AI

This kit is also everyday research code: Counter builds token frequency tables (the first step of BPE training), defaultdict(list) holds inverted indexes in retrieval prototypes, heapq drives beam candidates, and the aliased-rows gotcha has silently corrupted more than one attention-mask initialization in the wild. Interview fluency and notebook fluency are the same muscle.
