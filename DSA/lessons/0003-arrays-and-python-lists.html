Arrays & Python Lists Under the Hood

Today's skill: predict which list operations are O(1) and which are O(n) — by understanding what a Python list actually is.

Why this, now

Arrays are the substrate of everything else — strings, stacks, heaps, hash tables, and (in ML) tensors are all arrays underneath. Interviewers expect you to know, not guess, that append is cheap and insert(0, …) is not.

An array is contiguous memory

An array stores its elements side by side in one block of memory. That single fact explains everything:

index:    0     1     2     3     4
        ┌─────┬─────┬─────┬─────┬─────┐
memory: │ 17  │ 42  │  8  │ 99  │  3  │
        └─────┴─────┴─────┴─────┴─────┘
  • Index access is O(1): the machine computes address = start + i × slot_size — one multiplication, no searching. arr[500_000] costs the same as arr[0].
  • Middle insertion is O(n): to insert at position 0, every element must shift one slot right to keep the block contiguous. n elements → n moves.
  • Search by value is O(n): contiguity tells you nothing about where a value lives, so x in arr scans front to back.

Static vs. dynamic arrays

A static array (C, Java's int[]) has fixed capacity, chosen at creation. A dynamic array — which is what a Python list is — wraps a static array and resizes it when full:

  1. Keep a block bigger than needed (capacity > length).
  2. append writes into the spare slot: O(1).
  3. When full, allocate a new block roughly double the size, copy everything over (O(n)), continue.

The occasional O(n) copy is rare enough that averaged over many appends, each costs O(1) — called amortized O(1). In an interview, say: "append is amortized O(1)." That word signals you know about the resize.

The costs, derived

Mission link · ML/AI

A tensor is a contiguous array plus shape metadata — that's the whole trick. Contiguity is why GPUs can stream tensor data at full memory bandwidth, why reshape can be free (change metadata, not data), and why "memory layout" shows up in performance work. When PyTorch complains a tensor is "not contiguous," you now know exactly what it means.
