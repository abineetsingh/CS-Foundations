# Data Structures & Algorithms (Python) Resources

## Knowledge

- [Course: NeetCode — Data Structures and Algorithms for Beginners](https://neetcode.io/courses/dsa-for-beginners/0)
  Free video course covering exactly the DSA topics interviews require, in dependency order. Use for: the primary spine of this curriculum — watch the matching chapter alongside each lesson here.
- [Roadmap: NeetCode Roadmap + NeetCode 250 problem list](https://neetcode.io/roadmap)
  Dependency-ordered topic map with curated problems ([NeetCode 250](https://neetcode.io/practice/practice/neetcode250) is the beginner-friendly list). Use for: choosing which problems to drill after each topic is learned.
- [Book: _Problem Solving with Algorithms and Data Structures using Python_ (3rd ed.) — Miller & Ranum, free interactive](https://runestone.academy/ns/books/published/pythonds3/index.html)
  Free interactive university textbook; code runs in the browser. Use for: deeper theory than NeetCode gives — especially the "Analysis" chapter for Big-O and the chapters on trees/graphs.
- [Reference: Python Wiki — TimeComplexity](https://wiki.python.org/moin/TimeComplexity)
  The canonical table of CPython operation costs for list/dict/set. Use for: settling any "is this O(1) or O(n)?" question about a built-in operation.
- [Reference: Python Morsels — Python Big-O time complexities](https://www.pythonmorsels.com/time-complexities/)
  Friendlier annotated version of the above. Use for: quick lookups while solving problems.
- [Guide: Tech Interview Handbook — algorithms study cheatsheets](https://www.techinterviewhandbook.org/algorithms/study-cheatsheet/)
  Battle-tested interview strategy and per-topic cheatsheets. Use for: interview technique (how to talk through solutions), and per-topic edge-case checklists closer to interview time.
- [Video: freeCodeCamp — Data Structures and Algorithms in Python, full course](https://www.classcentral.com/course/freecodecamp-data-structures-and-algorithms-in-python-full-course-for-beginners-57034)
  ~13h beginner video course. Use for: an alternative explanation when a NeetCode chapter doesn't click.

## Wisdom (Communities)

- [r/leetcode](https://reddit.com/r/leetcode)
  Large, active community of interview preppers, including AI-lab candidates. Use for: interview experience reports (search "Anthropic interview", "OpenAI interview"), plateau troubleshooting, study-plan critique.
