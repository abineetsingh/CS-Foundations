Reflexes that score in interviews

  • Envelope drives design: compute QPS/storage, then state the verdict ("reads dominate → cache-first"). Numbers that don't change the design were wasted. (0006)
  • Declare budgets unprompted: staleness ("timelines 60 s stale, fine"), latency (p99 < 200 ms), error budget (99.9%). The strongest experience-signal. (0009, 0035)
  • Every tradeoff as "X because Y; alternative costs Z." Never a bare product name. (0040)
  • Truth vs derived: one source of truth; caches, search indexes, feeds, vector indexes are rebuildable projections fed async — losing them degrades, not destroys. (0020, 0043)
  • Idempotency is everywhere: retries, queues, event replay, agent tools all demand it. Ask of every handler: "safe to run twice?" (0022, 0025)
  • Pushback is direction: "what if a celebrity posts?" points at the intended deep dive. Welcome it. (0040)
  • Defer complexity, name the trigger: "shard when the primary nears limits" beats premature sharding. (0016, 0021)
