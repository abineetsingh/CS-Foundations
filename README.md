# CS Fundamentals and Foundations for Software Engineers

The goal of this repo is to provide a comprehensive and easy to digest set of curriculums to learn and master most commonly seen and used concepts in Software Engineering interviews. 
The lessons are single HTML pages that can be opened in any browser which explains a given conecpt, a brief learning plan and some short animations/quizzes for deeper understanding.
The material is deisnged to be consumed with spaced repitition approximately over a 3 month timeline to ingrain the lessons and conecpts for deeper mastery. 

Repo is structured in two core directories:

## Data Strucctures 

DSA
- CURRICULUM.md (the course curriculum broken down into phases)
- lessons directory 
- lesson0-n.html (open in browser to get the interactive lesson)
